How to Docs#

Building the Docs#

# go repo root folder
cd NavAbilitySDK.py

# Install required Python dependencies (Sphinx etc.)
pip install -r docs/requirements.txt

# Run Jupyter Book
jupyter-book build docs/

# View the docs with for instance firefox
firefox docs/_build/html/index.html
